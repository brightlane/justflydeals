# JustFly Deals USA — FlightDealsPro

**Live site:** https://brightlane.github.io/justflydeals/

USA flight affiliate site for JustFly.com targeting cheap flight, airline ticket, and travel deal keywords across all 50 states, 117 US origin cities, 30 domestic destinations, 38 international destinations, and 200 long-tail flight keywords.

## What this site does

Generates 10,000+ SEO pages covering every USA flight search angle. Every page matches the FlightDealsPro design (navy/orange, Inter font, urgency countdown, sticky CTA) from the original `index.html`.

- **State pages** — "cheap flights Alabama", "JustFly review Texas", "last minute flights Florida" — 50 states × 40 intents = 2,000 pages
- **City → Domestic route pages** — "flights from New York to Miami", "flights from Chicago to Las Vegas" — 117 US cities × 29 domestic dests = 3,393 pages
- **City → International route pages** — "flights from Houston to Cancún", "flights from LAX to Tokyo" — 117 US cities × 38 intl dests = 4,446 pages
- **Long-tail pages** — 200 research and informational pages
- **Static pages** — Flights, Destinations, Tips, Comparison, FAQ (5 pages)

## How it works

`build.py` runs daily via GitHub Actions and generates all HTML pages, `sitemap.xml`, `robots.txt`, and `llms.txt` from scratch, then commits and pushes to main. GitHub Pages deploys automatically on every push.

## Files

| File | Purpose |
|---|---|
| `build.py` | Main page generator — run this to rebuild the site |
| `.github/workflows/justfly-engine.yml` | Daily build trigger (1 AM UTC) |
| `.github/workflows/pages.yml` | GitHub Pages deployment |
| `index.html` | Homepage (original FlightDealsPro design) |
| `sitemap.xml` | Full sitemap — generated by build.py |
| `robots.txt` | Crawl rules — generated by build.py |
| `llms.txt` | LLM-readable site summary |
